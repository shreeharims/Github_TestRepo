Q8. Which field in the cart cookie can be modified to change item price?

Procedure
Add item to cart.

Extract cookie using:

<script>alert(document.cookie)</script>
Copy CartID value.
Decode using URL decoder.

Example:

1527258326:1:11.5:1000
Identify fields:
Field 1 → Timestamp
Field 2 → Unused
Field 3 → Price
Field 4 → Item ID
Modify Field 3 (price).
Refresh page and observe change.
Answer

Field 3 (Price Field)
